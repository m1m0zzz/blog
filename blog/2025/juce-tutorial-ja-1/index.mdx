この記事は [JUCE Advent Calendar 2025](https://qiita.com/advent-calendar/2025/juce) の12月13日の記事です。

以前より公開していた、[JUCE日本語版チュートリアル](https://m1m0zzz.github.io/juce-tutorial-ja/) の大規模改修を行いました。

## アップデート内容

- [オリジナルのチュートリアル](https://github.com/juce-framework/JUCE-tutorials) がGitHubで公開されたため、それをsubmoduleへ追加
- DeepL + 自力で翻訳していたものを、Claude を使い翻訳
- 壊れたリンクの修正
- 画像の直リンクを廃止
- OGP, favicon の更新

Getting Started 以外のすべてのページがClaudeによって翻訳されています。

## 所感

公開から1年以上経っても、残念な感じの翻訳がある状況でしたが、オリジナルのマークダウンが公開されたことで、楽に作業することができました。

マークダウンを公開してくれた、JUCEチームのみなさんと、Claude君に感謝です。

良ければ GitHub のスター押してくれると嬉しいです！

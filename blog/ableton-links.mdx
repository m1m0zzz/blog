## 公式 マニュアル

**Liveのキーボードショートカット**

https://www.ableton.com/ja/manual/live-keyboard-shortcuts/

作業効率が段違いなので、キーボードショートカットは覚えましょう！  
上のリンクに全部載っていますが、よく使うものを抜粋しておきます  

| | Windows	| Macintosh |
| --- | --- | --- |
| ブラウザ内を検索 | CTRL-F | CMD-F |
| MIDIクリップを挿入 | CTRL-Shift-M |	CMD-Shift-M |
| オーディオトラックを挿入 | CTRL-T | CMD-T |
| MIDIトラックを挿入 | CTRL-Shift-T | CMD-Shift-T |
| Quantize Settings… | CTRL-Shift-U | CMD-Shift-U |

Quantize Settings… で クオンタイズのかかり具合を調節する機能をよく使っています。
